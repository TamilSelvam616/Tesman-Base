HTML Tables மற்றும் Forms

1. HTML Tables

Tables data-வ row-கள் மற்றும் column-களாக organize பண்ணுது, schedule-கள், product list-கள், அல்லது comparison chart-களுக்கு perfect. முக்கிய tags: <table> table-ஐ hold பண்ணுது, <tr> row-களுக்கு, <th> header-களுக்கு (accessibility-க்கு scope உடன்), மற்றும் <td> data cell-களுக்கு. colspan மற்றும் rowspan attributes cell-களை column-கள் அல்லது row-களில் merge பண்ணுது.

<table>
    <thead>
        <tr>
            <th scope="col">Course</th>
            <th scope="col">Instructor</th>
            <th scope="col">Duration</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td colspan="2">HTML & CSS</td>
            <td>4 weeks</td>
        </tr>
        <tr>
            <td>JavaScript</td>
            <td>Jane Doe</td>
            <td>6 weeks</td>
        </tr>
    </tbody>
</table>

Output: Table

கீழே இருக்கும் table ஒரு course schedule-ஐ காட்டுது, "HTML & CSS" course-க்கு இரண்டு column-கள் (Course மற்றும் Instructor) merge ஆகி இருக்கு.

Course Instructor Duration
HTML & CSS 4 வாரங்கள்
JavaScript Jane Doe 6 வாரங்கள்

Visual Description: Table-ஓட background cream color-ல இருக்கு, gold border-கள் உடன். Header-கள் navy color-ல gold text-ஆ இருக்கு, even row-கள் light gold background-ல contrast-க்காக இருக்கு. "HTML & CSS" cell இரண்டு column-களை span பண்ணுது, Duration column-ல "4 வாரங்கள்" cell-ஓட align ஆகி இருக்கு. Text bold-ஆ இருக்கு, Playfair Display font use பண்ணுது.

2. HTML Forms

Forms user input-ஐ collect பண்ணுது, like registration or feedback. <form> tag form-ஐ define பண்ணுது, action (data எங்கே send ஆகுது) மற்றும் method (GET or POST) attributes உடன். <input>, <select>, மற்றும் <textarea> different data types-ஐ handle பண்ணுது, required மற்றும் pattern attributes validation-க்கு use ஆகுது.

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" placeholder="Enter your name" required>
    <label for="course">Course:</label>
    <select id="course" name="course">
        <option value="">Select a course</option>
        <option value="html">HTML & CSS</option>
        <option value="js">JavaScript</option>
    </select>
    <label for="feedback">Feedback:</label>
    <textarea id="feedback" name="feedback" rows="4" placeholder="Your feedback"></textarea>
    <button type="submit">Submit</button>
</form>

Output: Form

கீழே இருக்கும் form user-கள் name, course select பண்ண, மற்றும் feedback கொடுக்க allow பண்ணுது.

Visual Description: Form vertical-ஆ stack ஆகி இருக்கு, bold navy label-கள் மற்றும் cream color input field-கள் gold border-ஓட இருக்கு. Text input, dropdown, மற்றும் textarea consistent bold font-ஐ use பண்ணுது. Submit button gold color-ல navy text-ஆ இருக்கு, hover பண்ணும்போது light gold ஆகுது. Focus state-ல light gold border மற்றும் subtle glow இருக்கு. Layout clean-ஆ இருக்கு, mobile screen-களுக்கு fit ஆகுது.

3. Tables மற்றும் Forms Combine பண்ணது

Tables மற்றும் forms combine பண்ணி interactive data display create பண்ணுது, like editable tables for record manage பண்ண. ஒவ்வொரு table row-ல form இருக்கு data update பண்ணவும், dashboard அல்லது admin panel-ல use ஆகுது.

<table>
    <thead>
        <tr>
            <th scope="col">Student</th>
            <th scope="col">Course</th>
            <th scope="col">Grade</th>
            <th scope="col">Actions</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>John Doe</td>
            <td>HTML & CSS</td>
            <td>
                <form action="/update-grade" method="POST">
                    <input type="hidden" name="student" value="John Doe">
                    <input type="text" name="grade" value="A" pattern="[A-F]" required>
                    <button type="submit">Update</button>
                </form>
            </td>
            <td>
                <form action="/delete-student" method="POST">
                    <input type="hidden" name="student" value="John Doe">
                    <button type="submit">Delete</button>
                </form>
            </td>
        </tr>
        <tr>
            <td>Jane Smith</td>
            <td>JavaScript</td>
            <td>
                <form action="/update-grade" method="POST">
                    <input type="hidden" name="student" value="Jane Smith">
                    <input type="text" name="grade" value="B" pattern="[A-F]" required>
                    <button type="submit">Update</button>
                </form>
            </td>
            <td>
                <form action="/delete-student" method="POST">
                    <input type="hidden" name="student" value="Jane Smith">
                    <button type="submit">Delete</button>
                </form>
            </td>
        </tr>
    </tbody>
</table>

Output: Table + Form

கீழே இருக்கும் table ஒரு student roster-ஐ காட்டுது, ஒவ்வொரு row-ல grade update பண்ணவும், student delete பண்ணவும் forms இருக்கு.

மாணவர் Course Grade Actions
John Doe HTML & CSS
Jane Smith JavaScript

Visual Description: Table cream background-ல gold border-களோட இருக்கு, bold navy text மற்றும் navy header-கள் gold text-ஆ இருக்கு. Even row-கள் light gold-ல readability-க்காக இருக்கு. ஒவ்வொரு row-ல இரண்டு forms இருக்கு: ஒன்னு grade update பண்ண (A-F grades-க்கு restricted text input), இன்னொன்னு student delete பண்ண. Inputs மற்றும் buttons form section-ஓட consistent bold font-ஆ இருக்கு, gold border-கள் மற்றும் hover effect-களோட. Layout compact-ஆ இருக்கு, table cell-களுக்குள் forms neatly fit ஆகுது.
