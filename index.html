Learn HTML Basics

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the World Wide Web.

  • HyperText: Refers to text with links (hyperlinks) that connect to other web pages or parts of the same page, enabling non-linear navigation.
  • Markup: HTML uses special instructions called tags to define the structure and presentation of content, like headings or paragraphs.
  • Language: A standardized system that web browsers (e.g., Chrome, Firefox, Edge) understand to display web pages.

Purpose of HTML

HTML provides the foundation for web pages by organizing content such as text, images, links, and multimedia into a structure that browsers can render.

  • It defines the skeleton of a website, specifying where content appears and how it’s organized.
  • It works alongside CSS (for styling) and JavaScript (for interactivity) to create modern websites.

History of HTML

HTML was created by Tim Berners-Lee in 1990 as part of the World Wide Web. It has evolved significantly over time:

  • Early versions (HTML 1.0, 2.0) were simple, focusing on basic text and links.
  • HTML5, released in 2014, is the current standard, supporting multimedia (video, audio), animations, and interactive features.

How HTML Works

  • HTML files are text files saved with a .html extension (e.g., mywebsite.html).
  • These files are written using plain text and opened in web browsers to display content.
  • HTML is not a programming language but a markup language, meaning it focuses on structure, not logic or calculations.

Basic Structure of an HTML File

Every HTML page follows this basic setup:

<!DOCTYPE html>
<html>
<head>
    <title>Your Page Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
  • <!DOCTYPE html>: Declares the document as HTML5.
  • <html>: The main container for all HTML content.
  • <head>: Contains metadata, like the page title shown in the browser tab.
  • <body>: Holds visible content, such as text, images, or links.

Simple Example Page

Here’s a basic HTML page for your Tesman Learning website:

<!DOCTYPE html>
<html>
<head>
    <title>Tesman Learning</title>
</head>
<body>
    <h1>Welcome to Tesman Base</h1>
    <p>Learn new skills today!</p>
    <a href="https://tesman.com">Join Now</a>
    <img src="learning.jpg" alt="People studying">
    <ul>
        <li>Math Courses</li>
        <li>Science Lessons</li>
    </ul>
</body>
</html>

Key Tips

  • Save with .html: Use a text editor (like Notepad or VS Code) and save your file with a .html extension.
  • Test in a browser: Open your file in Chrome or Edge to see how it looks.
  • Comments: Use <!-- Comment here --> to add notes that don’t appear on the page.
