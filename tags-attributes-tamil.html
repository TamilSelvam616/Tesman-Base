HTML Tags & Attributes

1. Document Foundation

  • <!DOCTYPE html>: இது உங்க HTML file ஒரு HTML5 document-னு browser-க்கு சொல்லுது, மாதிரி ஒரு sign board. File-ஓட முதல் line-ல இது இருக்கணும். Case-sensitive இல்ல, so capital or small letters எழுதலாம். இது browser-ஐ correct-ஆ work பண்ண வைக்குது.
    Example:
    <!DOCTYPE html>
  • <html>: இது முழு HTML content-ஐ வச்சிருக்கும் பெரிய box மாதிரி. எல்லாம் இதுக்குள்ள தான் இருக்கும். Attributes like lang="en" language-ஐ set பண்ணுது (accessibility-க்கு help ஆகும்) and dir="rtl" text-ஐ right-to-left ஆக்குது, like Arabic language-க்கு.
    Example:
    <html lang="en"></html>

2. Metadata Masters

  • <head>: இது browser-க்கும் search engine-க்கும் instructions கொடுக்கும், ஆனா page-ல visible-ஆ தெரியாது. Title, meta tags, CSS links எல்லாம் இதுக்குள்ள இருக்கும்.
    Example:
    <head><title>My Page</title></head>
  • <title>: Page-ஓட name-ஐ browser tab-லயும் Google search-லயும் காட்டுது. Short-ஆ, clear-ஆ இருக்கணும்.
    Example:
    <title>Tesman Learning</title>
  • <meta>: Extra details கொடுக்குது. charset="UTF-8" எல்லா language characters-ஐ support பண்ணுது. name="viewport" mobile-ல page-ஐ சரியா fit பண்ணுது. name="description" Google-ல search snippet-ஆ வருது.
    Example:
    <meta charset="UTF-8">
<meta name="viewport" content="width=device-width">
<meta name="description" content="Learn coding">
  • <link>: External files-ஐ page-ஓட connect பண்ணுது, like CSS file for styling or favicon for browser tab icon.
    Example:
    <link rel="stylesheet" href="styles.css">
<link rel="icon" href="favicon.ico">

3. Content Organizers

  • <header>: Page-ஓட top part, like logo or menu. <footer>: Bottom part, like contact or copyright.
    Example:
    <header><h1>Tesman</h1></header>
<footer><p>Contact Us</p></footer>
  • <nav>: Navigation links வச்சிருக்கு, like menu bar. Screen readers இதை முக்கியமா பாக்குது.
    Example:
    <nav><a href="home.html">Home</a></nav>
  • <main>: Page-ஓட main content, ஒரு page-ல ஒரே main தான் இருக்கணும்.
    Example:
    <main><article>Content</article></main>
  • <article>: Independent content, like blog post or news article, தனியா stand பண்ணும்.
    Example:
    <article><h2>Blog Post</h2></article>
  • <section>: Related content-ஐ group பண்ணுது, like book chapter with heading.
    Example:
    <section><h3>Chapter 1</h3></section>
  • <div>: General box for grouping content, styling-க்கு use பண்ணலாம்.
    Example:
    <div><p>Text</p></div>
  • <span>: Small inline text-ஐ style பண்ண use ஆகுது, like one word highlight.
    Example:
    <span>Highlighted</span>

4. Text Hierarchy

  • <h1>-<h6>: Headings, <h1> biggest, <h6> smallest. One <h1> per page SEO-க்கு good. Screen readers navigation-க்கு use பண்ணுது.
    Example:
    <h1>Main Title</h1>
<h2>Subheading</h2>
  • <p>: Paragraphs for text blocks, automatic-ஆ space மேலயும் கீழயும் add ஆகுது.
    Example:
    <p>This is a paragraph.</p>
  • <strong>: Important text-ஐ bold ஆக்குது. <em>: Emphasized text-ஐ italic ஆக்குது. <mark>: Text-ஐ highlight (yellow) ஆக்குது.
    Example:
    <strong>Important</strong>
<em>Emphasized</em>
<mark>Highlighted</mark>

5. Media Elements

  • <img>: Images-ஐ page-ல show பண்ணுது. src image path சொல்லுது, alt accessibility-க்கு description கொடுக்குது, loading="lazy" page load fast ஆக்குது.
    Example:
    <img src="photo.jpg" alt="Scenery" loading="lazy">
  • <video> and <audio>: Video or audio play பண்ணுது. controls play/pause buttons add பண்ணுது, autoplay automatic play (browsers block பண்ணலாம்), loop repeat ஆக்குது.
    Example:
    <video src="vid.mp4" controls loop></video>
<audio src="song.mp3" controls></audio>

6. Navigation & Links

  • <a>: Links create பண்ணுது. href URL or section point பண்ணுது, target="_blank" new tab-ல open பண்ணுது (security-க்கு rel="noopener" use பண்ணுங்க), download file-ஐ download ஆக்குது.
    Example:
    <a href="https://tesman.com" target="_blank" rel="noopener">Visit</a>
<a href="file.pdf" download>Download</a>

7. Data Organization

  • <table>: Table create பண்ணுது. <tr>: Row. <th>: Header cell (scope="col/row" accessibility-க்கு). <td>: Data cell (colspan/rowspan merge பண்ணுது).
    Example:
    <table>
  <tr><th scope="col">Name</th></tr>
  <tr><td>John</td></tr>
</table>
  • <ul> and <ol>: Bullet list (unordered) or number list (ordered). <ol> attributes: reversed, start. <li>: List item.
    Example:
    <ul><li>Item 1</li></ul>
<ol start="2"><li>Step</li></ol>

8. User Input (Forms)

  • <form>: User input collect பண்ணுது, like login or survey. action data எங்க submit பண்ணணும்னு சொல்லுது, method="GET/POST" data send பண்ணும் way, form auto-fill control பண்ணுது.
    Example:
    <form action="/submit" method="POST">
  <input type="text">
</form>
  • <input>: Input fields create பண்ணுது. Types: text, email, password, checkbox, radio, file (accept=".pdf" file type restrict பண்ணுது), date, color, range.
    Example:
    <input type="email" required>
<input type="file" accept=".pdf">
  • Special Attributes: required field mandatory ஆக்குது, pattern="[A-Za-z]" input-ஐ validate பண்ணுது, placeholder hint text காட்டுது.
    Example:
    <input type="text" placeholder="Enter name" required>

9. Interactive Elements

  • <details> and <summary>: Click பண்ணா expand ஆகும் content box, JavaScript இல்லாம work ஆகும்.
    Example:
    <details>
  <summary>Click me</summary>
  <p>Hidden content</p>
</details>
  • <dialog>: Popup or modal window create பண்ணுது. open attribute or JavaScript (.showModal()) use பண்ணலாம்.
    Example:
    <dialog open>Popup content</dialog>

10. Global Attributes

  • Identification: id unique identifier, one per page. class reusable style groups.
    Example:
    <div id="unique" class="style">Content</div>
  • Accessibility: aria-label screen readers-க்கு description கொடுக்குது, tabindex keyboard navigation control பண்ணுது.
    Example:
    <button aria-label="Close" tabindex="0">X</button>
  • Custom Data: data-* JavaScript-க்கு custom info store பண்ணுது, like data-user-id.
    Example:
    <div data-user-id="123">User</div>

Key Concepts to Remember

  • Semantics Over Style: Tags-ஐ meaning-அடிப்படையில select பண்ணுங்க, looks-க்காக இல்ல.
    Example:
    <header><h1>Title</h1></header>
  • Attribute Order: Order technical-ஆ affect பண்ணாது, ஆனா alphabetical order common-ஆ இருக்கு.
    Example:
    <img alt="Photo" src="image.jpg">
  • Progressive Enhancement: CSS/JS இல்லாமயும் page work ஆகணும்னு ensure பண்ணுங்க.
    Example:
    <form action="/submit"><input type="text"></form>
  • Accessibility: Screen readers correct tags and attributes-ஐ depend பண்ணுது.
    Example:
    <button aria-label="Submit">Send</button>
