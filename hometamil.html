HTML Basics கத்துக்கோங்க

HTML என்ன பண்ணுது?

  • HTML browser-க்கு (like Chrome or Microsoft Edge) content-ஐ எப்படி organize பண்ணி display பண்ணணும்னு சொல்லுது.
  • இது tags use பண்ணி page-ஓட different parts-ஐ, like headings, paragraphs, or images, குறிக்குது.

HTML எப்படி வேலை செய்யுது?

  • HTML ஒரு text file-ல .html extension-ஓட (e.g., mywebsite.html) எழுதப்படுது.
  • Tags என்பது angle brackets-ல இருக்கும் instructions, like <p> for paragraph.
  • Most tags ஜோடியா வருது: opening tag (e.g., <p>) and closing tag (e.g., </p>). Content இவங்க நடுவுல இருக்கும்.

HTML File-ஓட Basic Structure

எல்லா HTML page-உம் இந்த basic setup-ஐ follow பண்ணுது:

<!DOCTYPE html>
<html>
<head>
    <title>Your Page Title</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>
  • <!DOCTYPE html>: இது HTML5 document-னு browser-க்கு சொல்லுது.
  • <html>: எல்லா HTML content-க்கும் main container.
  • <head>: Page-ஓட info, like browser tab-ல காட்டுற title, வச்சிருக்கு.
  • <body>: Text, images, or links போன்ற visible stuff-ஐ உள்ளடக்குது.

Simple Example Page

இதோ உங்க Tesman learning website-க்கு ஒரு basic HTML page:

<!DOCTYPE html>
<html>
<head>
    <title>Tesman Learning</title>
</head>
<body>
    <h1>Welcome to Tesman Base</h1>
    <p>Learn new skills today!</p>
    <a href="https://tesman.com">Join Now</a>
    <img src="learning.jpg" alt="People studying">
    <ul>
        <li>Math Courses</li>
        <li>Science Lessons</li>
    </ul>
</body>
</html>

Key Tips

  • .html-ஆ save பண்ணுங்க: Text editor (like Notepad or VS Code) use பண்ணி file-ஐ (something.html)-ஆ save பண்ணுங்க.
  • Browser-ல test பண்ணுங்க: File-ஐ Chrome or Edge-ல open பண்ணி எப்படி இருக்குனு பாருங்க.
  • Nesting: Tags உள்ள tags இருக்கலாம், ஆனா reverse order-ல close பண்ணுங்க (e.g., <p><strong>Bold text</strong></p>).
  • Case matter இல்ல: <P> and <p> same-ஆ வேலை செய்யும், ஆனா lowercase standard-ஆ இருக்கு.
  • Comments: <!-- Comment here --> use பண்ணி page-ல தெரியாத notes சேர்க்கலாம்.
